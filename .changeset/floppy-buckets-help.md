Add support for Astro 7 and drop support for Astro 4. The supported `astro` peer dependency range is now `^5.0.0 || ^6.0.0 || ^7.0.0`. If you are still on Astro 4, follow the [Astro upgrade guide](https://docs.astro.build/en/guides/upgrade-to/v6/) to upgrade your project before updating `@clerk/astro`.
