# Heading

A semantic heading component with typographic variants and color intents. Renders as an `<h2>` by default and supports font-size, font-weight, and color theming through the Mosaic design system.

## Usage

```tsx
import { Heading } from '@clerk/ui/mosaic/components/heading';

<Heading size="lg" intent="primary">
  Page Title
</Heading>
```

### Sizes

The `size` prop controls the font-size and line-height via the theme's typography scale:

```tsx
<Heading size="xs">Extra small heading</Heading>
<Heading size="sm">Small heading</Heading>
<Heading size="base">Base heading</Heading>
<Heading size="lg">Large heading</Heading>
<Heading size="xl">Extra large heading</Heading>
<Heading size="2xl">2XL heading</Heading>
```

### Intents

The `intent` prop controls the color:

```tsx
<Heading intent="primary">Primary heading</Heading>
<Heading intent="mutedForeground">Muted heading</Heading>
<Heading intent="destructive">Destructive heading</Heading>
```

### Custom Styling

Use the `sx` prop for custom Emotion styles:

```tsx
<Heading sx={{ marginBottom: 24 }}>
  Styled Heading
</Heading>
```

### Render Prop

Use the `render` prop for custom markup:

```tsx
<Heading render={props => (
  <h1 {...props}>
    Custom Heading
  </h1>
)} />
```

## Props

| Prop     | Type                                              | Default     | Description                   |
| -------- | ------------------------------------------------- | ----------- | ----------------------------- |
| `size`   | `'xs' \| 'sm' \| 'base' \| 'lg' \| 'xl' \| '2xl'` | `'base'`    | Font size variant             |
| `intent` | `'primary' \| 'mutedForeground' \| 'destructive'` | `'primary'` | Text color variant            |
| `sx`     | `CSSObject`                                       | —           | Custom Emotion styles         |
| `render` | `(props) => ReactNode`                            | —           | Render prop for custom markup |
| `ref`    | `Ref<HTMLHeadingElement>`                         | —           | Forward ref to the `<h2>`     |

All other HTML attributes of `<h2>` are supported via spread.
