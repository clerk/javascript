# Dialog

The styled Mosaic `Dialog` — an opinionated modal built on the headless `@clerk/headless`
dialog primitive. It owns a conventional layout (optional progress stepper, header,
description, body, a dismiss affordance, and an aligned action row) behind a single props API,
and inherits open state, portalling, focus management, dismissal (outside press / Escape), and
ARIA wiring from the primitive. For bespoke structure, drop down to the unstyled
[Dialog primitive](/primitives/dialog).

    { name: 'children', type: 'ReactNode' },
    { name: 'actions', type: 'DialogActions | ReactNode' },
  ]}
/>

`alignment`, `close`, `header`, and `description` are live knobs above; the rest are content
and behavior props. `trigger` is optional — omit it and drive the dialog through `open` / `onOpenChange` (e.g. from
a menu item elsewhere on the page). `actions` takes a structured `{ primary, secondary }` object
— primary is brand-colored, secondary uses the `secondary` `Button` color and dismisses the
dialog when it has no `onClick` — or arbitrary nodes (see [Custom actions](#custom-actions)).

## Examples

### Progress

Pass `progress` to render a stepper at the top of the dialog for multi-step flows; omit the prop
to hide it. `current` is 1-based.
