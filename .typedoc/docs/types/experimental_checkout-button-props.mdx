> **\_\_experimental_CheckoutButtonProps** = <code>\{ newSubscriptionRedirectUrl?: string; \}</code>

**`Experimental`**

This is an experimental API for the Billing feature that is available under a public beta, and the API is subject to change.

## See

/docs/billing/overview

It is advised to pin the SDK version and the clerk-js version to a specific version to avoid breaking changes.

## Example

```tsx
<ClerkProvider clerkJsVersion="x.x.x" />
```

## Properties

| Property                                                              | Type     | Description                                                                                                                    |
| --------------------------------------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------ |
| <a id="newsubscriptionredirecturl"></a> `newSubscriptionRedirectUrl?` | `string` | Full URL or path to navigate to after checkout is complete and the user clicks the "Continue" button. Defaults to `undefined`. |
