> **SDKMetadata** = <code>\{ environment?: string; name: string; version: string; \}</code>

## Properties

| Property                                | Type     | Description                                                                 |
| --------------------------------------- | -------- | --------------------------------------------------------------------------- |
| <a id="environment"></a> `environment?` | `string` | Typically this will be the `NODE_ENV` that the SDK is currently running in. |
| <a id="name"></a> `name`                | `string` | The npm package name of the SDK.                                            |
| <a id="version"></a> `version`          | `string` | The npm package version of the SDK.                                         |
