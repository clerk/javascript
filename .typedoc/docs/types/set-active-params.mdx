The parameters for the `setActive()` method.

## Properties

| Property                                  | Type                                                                                            | Description                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="beforeemit"></a> ~~`beforeEmit?`~~ | `BeforeEmitCallback`                                                                            | **Deprecated.** Use `redirectUrl` instead. Callback run just before the active session and/or organization is set to the passed object. Can be used to set up for pre-navigation actions.                                                                                                                                                                                                                                   |
| <a id="navigate"></a> `navigate?`         | `SetActiveNavigate`                                                                             | A custom navigation function to be called just before the session and/or organization is set. When provided, it takes precedence over the `redirectUrl` parameter for navigation. Example: `await clerk.setActive({ session, navigate: async ({ session }) => { const currentTask = session.currentTask; if (currentTask) { await router.push(`./onboarding/${currentTask.key}`) return } router.push('/dashboard'); } });` |
| <a id="organization"></a> `organization?` | <code>null \| string \| [OrganizationResource](/docs/references/javascript/organization)</code> | The organization resource or organization ID/slug (string version) to be set as active in the current session. If `null`, the currently active organization is removed as active.                                                                                                                                                                                                                                           |
| <a id="redirecturl"></a> `redirectUrl?`   | `string`                                                                                        | The full URL or path to redirect to just before the session and/or organization is set.                                                                                                                                                                                                                                                                                                                                     |
| <a id="session"></a> `session?`           | <code>null \| string \| [SignedInSessionResource](/docs/references/javascript/session)</code>   | The session resource or session ID (string version) to be set as active. If `null`, the current session is deleted.                                                                                                                                                                                                                                                                                                         |
